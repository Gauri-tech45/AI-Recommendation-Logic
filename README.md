# AI Recommendation Logic

## Overview

AI Recommendation Logic is a content-based recommendation system that suggests relevant movies, songs, books, and games based on user interests.

The project uses TF-IDF (Term Frequency-Inverse Document Frequency) and Cosine Similarity to analyze user preferences and generate personalized recommendations. It also includes cold-start handling for users with limited preference matches.

This project demonstrates practical applications of Artificial Intelligence in recommendation systems.

## Features
- AI-powered recommendation engine
- Content-based filtering
- TF-IDF feature extraction
- Cosine similarity scoring
- Personalized recommendations
- Multi-category recommendations
- Ranking and sorting pipeline
- Cold-start problem handling
- Interactive command-line interface

## Technologies Used
- Python 3
- Scikit-learn
- TF-IDF Vectorizer
- Cosine Similarity

## How It Works

### The recommendation system follows these steps:
- User interest collection
- Feature extraction using TF-IDF
- Vector representation of content
- Similarity calculation using cosine similarity
- Recommendation ranking
- Top-N recommendation filtering
- Cold-start detection and fallback recommendations

### The system generates recommendations across multiple categories:
- Movies
- Songs
- Books
- Games

## How to Run
1. Install the required library:
pip install scikit-learn

2. Run the project:
python recommendation_system.py
